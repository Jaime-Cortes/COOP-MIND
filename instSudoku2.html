SUDOKU

Para completar el SUDOKU basta con llenar las casillas restantes con los números del 1 al 16. Para ello se deben seguir 3 simples reglas.
1. El numero no se debe repetir rn toda la fila
2. El numero no se debe repetir en toda la columna
3. El numero no se debe repetir en toda la cuadricula de 4x4

El SUDOKU ayuda a ejercitar el córtex prefontal del cerebro, además de una mejor concenctración. En la parte central se encuentra un contador, si llega a 2000 pierdes. También hay 2 botones, uno para repetir el sudoku y otro para combrobar si los números que escribiste son correctos, caso contarrio la casilla se marcará de color rojo para indicar un error
